Babel - это инструмент для транспиляции кода JavaScript из одной версии стандарта в другую.
Основная цель Babel - обеспечить совместимость нового кода JavaScript с более старыми версиями JavaScript,
которые могут поддерживаться браузерами и другими средами выполнения.

Babel позволяет разработчикам использовать новые функции и возможности языка JavaScript,
которые могут быть еще не поддерживаемыми в текущей версии браузера или другой среды, и автоматически
преобразовывать этот код в эквивалентный код, который будет работать в этих средах.
